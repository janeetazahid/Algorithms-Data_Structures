The purpose of this lab was create a statemachine based on the state diagram given. This program starts by printing the current state, the user
can input 0 or 1 to see the next state based on the input values. the user can change the current state using the chnage commnad and print the 
state machine configurations using the print command. The delete command deletes a state and the garbage commmand shows any states which are unreachable.
